1. How to Install the full vim package:

a: Install vim with the command: sudo apt-get install vim

2. How to start vim to edit a new or existing file:

a: vim filename

3. How to get help in vim

a: vim --help

4. How to start the vim tutorial

a: type: vimtutor at the bash prompt

5. How to switch between Command (Normal) Mode, Insert Mode and Command Line Mode

a: command mode: ctrl+C and type / to type command Insert Mode: typr i Command Line mode: type :wq to save file and quit file or :q! to quit file without saving

6.How to save a file in vim

a: ctrl+c or esc to quit insert mode and type :w to save or type :wq to save and quit

7. How to exit vim

a: type :wq to save and quit or :q! to quit without saving

Include a brief table of navigation and editing commands

a: in normal mode, type those command line: k – navigate upwards j – navigate downwards l – navigate right side h – navigate left side H – Go to the first line of current screen. M – Go to the middle line of current screen. L – Go to the last line of current screen. ctrl+f – Jump forward one full screen. ctrl+b – Jump backwards one full screen ctrl+d – Jump forward (down) a half screen ctrl+u – Jump back (up) one half screen x - to delete the unwanted character u - to undo the last the command and U to undo the whole line CTRL-R to redo A - to append text at the end :wq - to save and exit :q! - to trash all changes dw - move the cursor to the beginning of the word to delete that word 2w - to move the cursor two words forward. 3e - to move the cursor to the end of the third word forward. 0 (zero) to move to the start of the line. d2w - which deletes 2 words .. number can be changed for deleting the number of consecutive words like d3w dd to delete the line and 2dd to delete to line .number can be changed for deleting the number of consecutive words
